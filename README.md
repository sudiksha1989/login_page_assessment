## Requirements

- [Node.js](https://nodejs.org/en/) 6+
- [mongoDB.js](https://www.mongodb.com/)

```shell
npm install
```


## Running

Add a `config.js` file in the `config` folder with mongodb port.

Production mode:

```shell
npm start
```

Development (Webpack dev server) mode:

```shell
npm run start:dev
```
